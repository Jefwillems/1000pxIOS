# 1000px IOS app

## Installation (Mac)
the project.sh file checks if cocoapods is installed and installs the projects dependencies.

    $ git clone git@github.com:Jefwillems/1000pxIOS.git
    $ cd 1000pxIOS
    $ sh ./project.sh
    now open the ProjectIOS.xcworkspace file
